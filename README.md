# BDMS Frontend

Production-grade Blood Donation Management System frontend built with React + Vite.

## Stack
- React
- Vite
- Tailwind CSS
- React Router v6
- Axios
- Zustand
- React Hot Toast

## Setup
1. Install dependencies
   npm install
2. Create environment file
   copy .env.example .env
3. Run development server
   npm run dev
4. Build for production
   npm run build

## API Base URL
Set `VITE_API_BASE_URL` to your Spring Boot backend endpoint, e.g. `http://localhost:8080/api`.
